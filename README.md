## 🌟 لماذا "عفّة"؟
التطبيقات الأجنبية لا تفهم ثقافتنا ولا تحتوي على محفزات دينية. وتطبيقات تتبع العادات العادية لا تصلح لطبيعة هذه المعركة. 
**عفّة** تجمع بين:
- 🧠 **العلم:** تتبع المحفزات وتحليلها (CBT).
- 🤲 **الروح:** أذكار، آيات قرآنية، وأدعية للثبات.
- 🔒 **الخصوصية:** البيانات مخزنة محلياً فقط على جهازك (بدون سيرفرات خارجية).

## ✨ أبرز المميزات

### 🏆 نظام إنجازات تفاعلي
- 10 مراحل متدرجة (من "البداية" إلى "سنة حرية").
- شريط تقدم دائري أنيق يتغير لونه حسب مستواك.
- تنبيهات احتفالية عند تحقيق كل إنجاز جديد مع آيات قرآنية.

### 🚨 زر الطوارئ (SOS)
- زر عائم (FAB) متاح دائماً عند الحاجة الملحة.
- تمرين تنفس (4-4-6 ثوان) لتهدئة الجهاز العصبي فوراً.
- آيات وأذكار تُعرض بشكل عاجل لتشتيت الانتباه.

### 📊 تتبع وتحليل المحفزات
- تسجيل الانتكاسات مع تحديد السبب (ملل، توتر، سهر، تصفح عشوائي).
- رسوم بيانية (Bar Charts) لتوضيح توزيع المحفزات.
- معرفة "أكثر محفز" يؤدي للانتكاسة لتجنبه مستقبلاً.

### 📖 محتوى روحي مخصص
- أذكار الصباح والمساء.
- أدعية صادقة للثبات والعفاف (مع ذكر المصادر).

### 🎨 تصميم مريح للعين
- ثيم داكن إسلامي (درجات اللون الأخضر) لتقليل الإرهاق البصري.
- تصميم نظيف يقلل من المشتتات.

## 🛠️ التقنيات المستخدمة
- **لغة البرمجة:** Kotlin 100%
- **واجهة المستخدم:** Jetpack Compose (Material Design 3)
- **البنية الهندسية:** MVVM (Model-View-ViewModel)
- **قاعدة البيانات:** Room Database (لتخزين سجل الانتكاسات)
- **التنقل:** Navigation Compose
- **التخزين المؤقت:** SharedPreferences (لمدة بداية التعافي وإعدادات التذكير)
- **الإشعارات:** AlarmManager + BroadcastReceiver (للتذكيرات اليومية)

## 📁 البنية الهندسية
المشروع مقسم بشكل نظيف لضمان سهولة التطوير:
```text
com.example.iffah/
├── data/            # (Room, Entity, DAO, Achievement Data)
├── ui/
│   ├── screens/     # (AchievementsScreen, StatsScreen, AdhkarScreen, SettingsScreen)
│   ├── components/  # (مكونات مخصصة)
│   └── theme/       # (الألوان والثيم الداكن)
└── navigation/      # (مسارات الشاشات)
```

## 🚀 كيفية تشغيل المشروع محلياً
1. تأكد من تثبيت أحدث إصدار من **Android Studio** (Iguana أو أحدث).
2. استنساخ المستودع:
   ```bash
   git clone https://github.com/eng-admery/Iffah-App.git
   ```
3. افتح مجلد المشروع في Android Studio.
4. انتظر حتى ينتهي Gradle من المزامنة (Sync).
5. شغّل التطبيق على جهاز حقيقي أو محاكي.
